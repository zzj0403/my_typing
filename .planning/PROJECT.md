# My Typing - 个人打字练习工具

## What This Is

一个轻量级的中文全拼打字练习工具。用户可以导入诗词、名人名言或自定义文章，看着原文逐句打字，获得实时的正确/错误反馈。

专为想要通过整篇文章练习中文打字的用户设计，区别于单字练习工具。

## Core Value

**能导入自己的文章 + 实时反馈 + 打完一篇。** 简单直接，够用就好。

## Context

**源项目**: https://github.com/yunsii/pinyin (Fork)
- 技术栈：TypeScript + Vite + React + Ant Design
- 功能：拼音打字练习，支持全拼和双拼
- 已有：拼音匹配逻辑、键盘提示、实时数据分析

**痛点**:
- 参考项目只能单字练习，无法整篇文章连续打
- 无法导入自定义文章

**用户期望**:
- 用自己喜欢的内容（诗词、名言）练习
- 看着原文打，获得即时反馈
- 打完一篇有成就感

## Constraints

- **Tech Stack**: TypeScript + Vite（参考项目技术栈）
- **输入法**: 仅全拼
- **内容格式**: 纯中文，保留标点
